## Prérequis

npm 8 (node 18)

## Installation

Pour installer l'application placez vous dans le repertoire nommé my-movie-will-go-on et lancer la commande suivante

```
npm install
```

Toujours dans le même répertoire lancez l'application avec cette commande

```
npm run serve
```
